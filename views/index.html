Install v1.6.0 Getting Started
  • Quick Start
  • Template Parsing
  • Official demo

Download and install 


  go get github.com/astaxie/beego

Create file hello.go


  package main

  import "github.com/astaxie/beego"

  func main() {
    beego.Run()
  }

Build and run 


  go build -o hello hello.go
  ./hello

View effects，Open your browser and visit http://localhost:8080

Congratulations! You just built your first beego app. Please see Documentation for going further.
