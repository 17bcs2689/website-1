BeeGo make it fun and easy to code more advanced web applications while still adhering to the MVC concepts!

Quick Start

Download and install beego: 

go get github.com/astaxie/beego

Download bee tool

Create new beego project in your GOPATH:

bee new website

Run your beego project:

bee run project:

Open your browser and visit http://localhost:8080
Congratulations! You just built your first beego app. Please see Documentation for going further.
